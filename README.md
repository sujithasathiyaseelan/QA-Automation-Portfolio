E-Commerce Automation Framework
Selenium | Java | Cucumber BDD

Project Overview

This project is a robust automation framework designed to test the end-to-end functionality of an e-commerce application. It was developed to demonstrate my proficiency in modern Automation QA practices after completing formal training.

Tech Stack

Language: Java 
Automation Tool: Selenium WebDriver 
Framework: Cucumber BDD (Behavior Driven Development) 
Build Tool: Maven 
Testing Engine: TestNG 
Pattern: Page Object Model (POM) 

Key Features

BDD Approach: Uses Gherkin feature files for readable, business-focused test scenarios.
Maintainability: Implemented Page Object Model (POM) to separate page-specific logic from test scripts.
Data Validation: Basic SQL integration to verify backend data consistency.
Reporting: Automatic generation of test execution reports via TestNG.
